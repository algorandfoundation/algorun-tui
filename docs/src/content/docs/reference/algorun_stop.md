## Synopsis                                           
                                                                                                  
                                                                                                  
Stop the node daemon                                                                              
                                                                                                  
Overview:                                                                                         
Stops the Algorand daemon on your local machine. Optionally, the daemon can be forcefully stopped.
                                                                                                  
This requires the daemon to be installed on your system.                                          

```
algorun stop [flags]
```
